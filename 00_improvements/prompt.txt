The Tennis Ranking List web application is used to manage tennis players and organize challenges between them.
In addition to displaying the current ranking list, the application offers functions for dynamically updating
the ranking positions and for managing player blocking and availability.

Backend: Python with the Flask framework
Real-time communication: Flask-SocketIO
Frontend: HTML, CSS, JavaScript (jQuery, Bootstrap)
Database: SQLite
JSON data for initializing the players (initial_players.json)

Please always ask questions if something is unclear.
